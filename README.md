# SLA-MVP-V2
**Story Loom AI MVP Version 2**

## Installation

1. Install Plugins on Unreal Engine (This version is based on Unreal Engine 5.3)
   - Blueprint Http Server, Blueprint File SDK, Web Communication, Json Blueprint Utilities, Movie Render Queue

2. Import CH_Detective Character (Metahuman + Paragon)
   - Asset Link - https://drive.google.com/file/d/1RAuBaDMwzZCWRvERxpTbkQwkvdDw-zsK/view
   - Extract to Game\Content folder

3. Import CH_RobotCop Character (Paragon)
   - Asset Link - https://drive.google.com/file/d/1yENmM6hVMYLv2e-VU_3HrBUYFfaXLAo9/view
   - Extract to Game\Content folder

4. Import LV_CyberCity (Location)
   - Asset Link - https://drive.google.com/file/d/1a9FxZ0DNu51n8vCdozcPpKoMKJemevVJ/view
   - Extract to Game\Content folder
